# Sistema de Gestión de Clientes

Aplicación de consola en Java para gestionar clientes mediante operaciones CRUD.

## Funcionalidades

- Agregar cliente
- Listar clientes
- Actualizar cliente
- Eliminar cliente
- Buscar cliente por ciudad

## Tecnologías utilizadas

- Java
- Programación Orientada a Objetos
- ArrayList
- Manejo básico de excepciones

## Ejecución

Ejecutar la clase `SistemaClientes` desde el IDE.
